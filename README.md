# Inscope Business Card

An offline, single-file business card editor for **Inscope Electronics Pvt. Ltd.**

Open `index.html` in a browser, edit the details, and download a print-ready PDF — both the front and back, in vector format.

## Use
1. Open `index.html` (Chrome recommended).
2. Edit **Name, Title, Phone, Email, Address**. The logo, website and company name are fixed brand elements.
3. Toggle **Trim / Bleed** to preview.
4. Download:
   - **No bleed** — trim size 55 × 85 mm.
   - **With bleed + crop marks** — 61 × 91 mm (3 mm bleed); use this when sending to a printer.

## Notes
- Works fully offline — no internet, libraries or plugins.
- Output is a vector, editable PDF (both sides).
- Long names/emails auto-fit to the card.

## Brand
- Lime `#A1FF08` · Black · Typeface: Work Sans
